Without the vast amount of freely available resources and open source projects on the internet today, the development of my blog would not have been possible. A big thank you to all contributors! The list below can never be complete especially if one wants to consider transient dependencies as well. Nevertheless I tried to list the most important resources which enabled the creation of this blog!

Resource Description Link Licence
Hugo Static site generator that powers this site https://gohugo.io/ Apache 2.0
Bulma CSS framework which is used as the site's theme https://bulma.io/ MIT
jquery Javascript library for HTML manipulation and event handling https://jquery.com/ MIT
Lunr Javascript search engine which provides full text search capability https://lunrjs.com/ MIT
sharer.js Javascript library to share the posts on social media https://ellisonleao.github.io/sharer.js/ GPL-3.0
mark.js Javascript library to highlight the found search terms in the document https://markjs.io/ MIT
timeago.js Javascript library to convert timestamps into relative time. https://timeago.org/s MIT
fontello Generator to embed the icon fonts in a single CSS file https://fontello.com/ MIT
Font Awesome Icon fonts by Dave Gandy https://fontawesome.com/ SIL OFL 1.1

The images are from Unsplash or Pixapay and are attributed directly with the author's name.
